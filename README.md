## 🌿 What's this?

**Xray-core** is a powerful proxy core, but it only lets you manage users through static configuration or gRPC API — neither of which plays nicely with an existing backend or your own business logic

**cnap-core** adds a simpler option: point it at your own HTTP API, and let your backend decide who gets in — no synchronization or workarounds required!

## 🐒 Status

Nothing is ready yet — architecture is still being designed!
